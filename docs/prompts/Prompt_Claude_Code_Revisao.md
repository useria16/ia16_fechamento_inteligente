# Prompt de revisão para Claude Code

Revise a implementação atual do projeto iA16 Fechamento Inteligente.

## Pontos obrigatórios de revisão

1. Verificar se todos os objetos customizados do banco estão em português.
2. Verificar se as migrations seguem padrão em português.
3. Verificar se não existe cadastro público de usuário.
4. Verificar se SQLAlchemy 2.x está sendo usado corretamente.
5. Verificar se SQLModel não foi usado.
6. Verificar se o frontend está separado do backend.
7. Verificar se Supabase Auth, Storage e Postgres estão configurados de forma coerente.
8. Verificar se não foram adicionadas funcionalidades fora do MVP.
9. Verificar se variáveis sensíveis não estão versionadas.
10. Verificar se o Docker Compose está coerente para VPS.

## Saída esperada

Entregue uma análise com:

- problemas encontrados
- risco de cada problema
- arquivos afetados
- recomendações
- correções sugeridas
- ordem recomendada de ajuste
