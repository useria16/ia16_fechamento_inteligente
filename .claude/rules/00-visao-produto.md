# Visão de produto

O iA16 Fechamento Inteligente é uma aplicação multicliente para conciliação financeira configurável.

Não é um ERP.

O MVP deve resolver:

- upload de planilhas financeiras
- criação de fechamento financeiro
- normalização dos dados
- conciliação baseada em regras
- identificação de divergências
- registro de observações
- exportação de planilha final
- dashboard simples
- logs de processamento

A Daxx Omnimedia é o primeiro cliente piloto, com conciliação bancária.

O segundo cliente usará conciliação de caixa, recebíveis e controles financeiros.

A aplicação deve usar o mesmo núcleo para todos os clientes.
